## 一、vue 和 react 的优缺点！

React和Vue相当于汽车自动挡和手动挡的区别

Vue:自动挡，入门门槛低，方便，省心，缺点是官方依赖度高，技术自由度相对较低
React:手动挡，自由度高，注重技术机械驾驶体验感，缺点就是入门门槛高
